# AI Voice Assistant

A simple Python voice assistant with a clean dark GUI.

## Install

```
py -3.12 -m pip install pyttsx3 SpeechRecognition pyaudio setuptools
```

If pyaudio fails:
```
py -3.12 -m pip install pipwin
py -3.12 -m pipwin install pyaudio
```

## Run

```
py -3.12 main.py
```

## What it can do

- Open apps — "open notepad", "open chrome", "open spotify"
- Open websites — "open youtube", "open gmail", "open whatsapp"
- Web search — "search python tutorials"
- Weather — "weather in Delhi"
- Time and date — "what time is it", "what is today's date"
- Jokes — "tell me a joke"
- Any new app installed — "open valorant", "open minecraft"
